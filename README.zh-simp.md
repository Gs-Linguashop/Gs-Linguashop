# G的语言小站

**语言:** [English](README.md) | [**简体中文**](README.zh-simp.md) | [繁體中文](README.zh-trad.md)

我是一个汉语语音学、草书书写与实验性字体系统的交叉领域爱好者。我的工作涵盖字体设计、开源学习资源，以及基于音符/音位的输入法和工具。

## 热点项目

- **[明鉴草书黑体](https://github.com/Gs-Linguashop/MingjianCaoshuHeiti.git)**  
  开源的草书黑体字体（开发中）。探索草书数字化的先驱与领先者（个人观点）。适合教学、草书数字化及日常使用。

- **[嗷呜草书学习资料](https://github.com/Gs-Linguashop/cursive-chinese-learn)**  
  开源草书中文学习资源，每周更新课程教案和教学视频。专注于系统化理解，方便草书学习者使用。

- **[谐声汉字 (Sino-Phonographics)](https://github.com/Gs-Linguashop/sino-phonographics.git)**  
  基于汉字音韵学，将形声字的声符进行字母化的书写系统。简单来说，就是把形声字去掉表意部首而使用。从数以万计的汉字提取千余个声符组成音节字母表，汉字使用者稍微适应即可使用。附有配套的工具可以将任意字体进行上述修改。

## 其他项目

- **[Rime-Gryihleng](https://github.com/Gs-Linguashop/rime-gryihleng.git)** – 上古汉语语音输入法，含全拼/双拼。
- **[谐声上古音输入法](https://github.com/Gs-Linguashop/phonograph-type)** – 基于汉字声符部件的实验性字形输入法。
- **[汉字声符字母表](https://github.com/Gs-Linguashop/phonograph-alphabet)** – 提取汉字声符，形成适用于其他语言的通用音节字母表。

# G的語言小站

**語言:** [English](README.md) | [简体中文](README.zh-simp.md) | [**繁體中文**](README.zh-trad.md)

我是一個漢語語音學、草書書寫與實驗性字體系統的交叉領域愛好者。我的工作涵蓋字體設計、開源學習資源，以及基於音符/音位的輸入法和工具。

## 熱點項目

- **[明鑑草書黑體](https://github.com/Gs-Linguashop/MingjianCaoshuHeiti.git)**  
  開源的草書黑體字體（開發中）。探索草書數位化的先驅與領先者（個人觀點）。適合教學、草書數位化及日常使用。

- **[嗷嗚草書學習資料](https://github.com/Gs-Linguashop/cursive-chinese-learn)**  
  開源草書中文學習資源，每週更新課程教案和教學視頻。專注於系統化理解，方便草書學習者使用。

- **[諧聲漢字 (Sino-Phonographics)](https://github.com/Gs-Linguashop/sino-phonographics.git)**  
  基於漢字音韻學，將形聲字的聲符進行字母化的書寫系統。簡單來說，就是把形聲字去掉表意部首，替代原本的漢字使用。從數以萬計的漢字提取千餘個聲符組成音節字母表，漢字使用者稍微適應即可使用。附有配套的工具可以將任意字體進行上述修改。

## 其他項目

- **[Rime-Gryihleng](https://github.com/Gs-Linguashop/rime-gryihleng.git)** – 上古漢語語音輸入法，含全拼/雙拼。  
- **[諧聲上古音輸入法](https://github.com/Gs-Linguashop/phonograph-type)** – 基於漢字聲符部件的實驗性字形輸入法。  
- **[漢字聲符字母表](https://github.com/Gs-Linguashop/phonograph-alphabet)** – 提取漢字聲符，形成適用於其他語言的通用音節字母表。
